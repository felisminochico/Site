História do Desenvolvedor

Este é um site estático desenvolvido por três linguagens: HTML, CSS e JavaScript. O site foi desenvolvido por mim(Felismino Chico), com o objetivo de atender as minhas necessidades, que são: aperfeiçoar os meus conhecimentos em Programação, desenvolvimento Web e Tecnologias básicas. E foi com isso que decidi desenvolver meu próprio Site para guardar todos os meus Projetos de aprendizado.

O meu primeiro contacto com as linguagens de desenvolvimento Web aconteceu no meu penúltimo ano do ensino médio. Foi aí que decidi aprender a desenvolver sites começando com as linguagens HTML e CSS. Logo após tornar-me um pouco experiente nas linguagens HTML e CSS, decidi partir no aprendizado da linguagem JavaScript, que é a única considerada uma linguagem de programação neste trio. Com o conhecimento deste trio, a minha paixão por programação e desenvolvimento Web foi aumentando cada vez mais. Hoje, encontro-me estudando uma outra linguagem de programação que é o PHP, com objetivo de atender a este meu site estático pra passar a ser dinâmico.

Foto do Desenvolvedor

Antes de qualquer interação com as linguagens acima citadas, eu havia estudado Lógica de Programação mais conhecido como Algoritmos, é um aprendizado que serve para dar os primeiros passos pra quem quer mergulhar no mundo da Programação como eu mergulhei e já nem quero mais sair deste mundo.

Desenvolvimento

Atualmente, estou desenvolvendo um Site semelhante a este para que possa atender várias pessoas no mundo que se encontram com as mesmas necessidades que as minhas. Desejo um dia ser um Desenvolvedor Web reconhecido pelo mundo como os grandes desenvolvedores que tínhamos e que temos hoje em dia.

Desenvolver Site ou programar são práticas que tornam o meu dia-a-dia muito melhor e tão confiante, que um dia eu possa desenvolver algo que muitas pessoas usarão este mesmo projecto.

Mas contudo, todos os projectos meus encontrados neste site, servirão de recordação para mim, para que um dia eu possa dizer Finalmente consegui.
